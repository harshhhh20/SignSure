04

Cryptography Concepts

Key concepts demonstrated in this project.

SHA-256
Produces a fixed 256-bit (32-byte) hash from any input. A single character change produces a completely different hash — the avalanche effect.
Hash Function
RSA-2048
Asymmetric cryptography. The private key signs; the public key verifies. 2048-bit key size makes brute-force computationally infeasible.
Asymmetric Encryption
Digital Signature
Provides three guarantees: Authenticity (who signed it), Integrity (not tampered), and Non-Repudiation (signer can't deny it).
Security Property

About

SignSure demonstrates digital signatures using OpenSSL, RSA-2048, and SHA-256. The platform allows users to sign documents, verify signatures, and detect tampering — all locally, with no cloud services.
